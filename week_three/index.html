CSS Box Model & Selectors

Introduction to the Box Model

Every HTML element is a box. This box consists of content, padding, borders, and margins.

This page serves as a simple demonstration of how CSS selectors and the box model work together to style and structure a web page. We'll be using different selectors to target specific elements.
